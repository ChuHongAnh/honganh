WritePad Editor

WritePad í an editor application taht allows you to create Web pages easily and quickly.

Steps to open WritePad editor:

  1. Click Start
  2. Click Programs
  3. click WritePad

Advantages of using a WritePad editor:

  • It allows faster creation of Web pages.
  • It allows easier updation of text or images in WritePad editor
  • Is is platform independent

disadvantages of using a WritePad editor

  • It consumes more memory space
  • It requires to fill-in more details such as data for each page
